Web hosting

Hospedagem na Web

English:
Web hosting is the term that is synonymous with website hosting and the term that covers all services in the area. In this way, the term used can be considered in the services that include the website hosting market.
Português:
Web hosting é o termo em inglês que é sinônimo de hospedagem de sites e o termo que abrange todos os serviços da área. Desta forma, o termo usado pode fazer referência geral aos serviços que contemplam o mercado de hospedagem de sites.

  • Date: January 2017

Host

hospedeiro

English:
host is a machine or device that uses the network with an IP address. In the hosting area of ​​the site, the Host can also be the same as a hosting provider, since it is responsible for all the structure and machinery for a transmission to take place and the reception of data among the others.
Português:
host é uma máquina ou um dispositivo que funciona conectado à rede com um endereço IP. Na área de hospedagem de site, um Host também pode ser o mesmo que um provedor de hospedagem, pois é ele quem fornece toda a estrutura e maquinário para que ocorra a transmissão e o recebimento de dados entre vários outros dispositivos.

  • Date: January 2017

Uptime

Tempo De Atividade

English:
This is a term that represents the uptime of a server or a central service of its activities without interruption. Most hosting companies use a marketing and sales machine because this service serves to show how good the service is.
Português:
Esse é um termo que representa o tempo de atividade que um servidor ou qualquer outro sistema executa suas atividades sem interrupção. A maioria das empresas de hospedagem usa esse fator como uma ferramenta de marketing e vendas, pois este atributo serve de termômetro para mostrar o quão bom é o serviço ofertado.

  • Date: January 2017

DNS

Sistema de Nome de Domínios

English:
The acronym DNS (Domain Name System) functions as an IP address translator. When we access a site, this system (the DNS) is responsible for translating the sequence of numbers that compose the IP into an intelligible domain name.
Português:
A sigla DNS (Sistema de Nome de Domínios) funciona como um tradutor de endereços IP’s. Quando acessamos um site, esse sistema (o DNS) se encarrega de traduzir a sequência de números que compõe o IP em um nome de domínio inteligível.

  • Date: January 2017

Data center

Centro de Dados

English:
A data center is a database site that processes high volumes of data and information. Data centers are responsible for the operation of telecommunications services and services essential to society. In general, that is what factors are allocated in specific environments and with robust systems of security, beyond of sitour less re-stricament.
Português:
Um data center é um espaço ou um local destinado a abrigar diversos equipamentos que processam altos volumes de dados e informações. Os data centers são responsáveis por manter o funcionamento de diversos serviços como telecomunicações e outros serviços essenciais para a sociedade. Em geral, esses equipamentos ficam alocados em ambientes específicos e com sistemas de segurança robustos, além de possuir acesso altamente restrito.

  • Date: January 2017

CPanel

CPanel

English:
CPanel is one of the best-known hosting control panels in the world. It serves to administer the main functionalities of a website hosting service and is usually already made available by the provider after hiring.
Português:
O cPanel é um dos painéis de controle de hospedagem mais conhecido do mundo. Ele serve para administrar as principais funcionalidades de um serviço de hospedagem de site e geralmente já é disponibilizado pelo provedor após a contratação.

  • Date: January 2017

VPS

VPS

English:
A Virtual Private Server (VPS) is a type of hosting service where a server is partitioned into other small servers that act with independent resources. This type of service is not a dedicated server, but is much more flexible and powerful than shared hosting.
Português:
Um VPS (Virtual Private Server) é um tipo de serviço de hospedagem em que um servidor é particionado em outros pequenos servidores que atuam com recursos independentes. Esse tipo de serviço não é um servidor dedicado, mas é muito mais flexível e poderoso do que o de hospedagem compartilhada.

  • Date: January 2017

CMS

Content Management System

English:
CMS is the acronym for Content Management System and refers to open programs or free softwares that are developed jointly by developers around the world.
Português:
CMS é a sigla de Content Management System e se refere a programas abertos ou softwares livres que são desenvolvidos em conjunto por desenvolvedores do mundo todo.

  • Date: January 2017

FTP

Protocolo de Transferência de Arquivos

English:
The acronym FTP stands for File Transfer Protocol. It is a free tool used to transfer files to your hosting server.
Português:
A sigla FTP significa File Transfer Protocol(Protocolo de Transferência de Arquivos). É uma ferramenta gratuita que serve para transferir arquivos para o seu servidor de hospedagem.

  • Date: January 2017

SSL

Secure Socket Layer

English:
A Secure Socket Layer (SSL) certificate serves to ensure the exchange of data on the Internet. The certificates guarantee the identity and authenticity between users and entities. This type of technology (encryption) is required to establish secure connections in the web environment and can be obtained for free or for a low cost.
Português:
Um certificado SSL serve para assegurar a troca de dados na Internet. Os certificados garantem a identidade e autenticidade entre usuários e entidades. Esse tipo de tecnologia (criptografia) é necessária para estabelecer conexões seguras no ambiente web e pode ser obtida de maneira gratuita ou por um baixo custo.

  • Date: January 2017

SSH

Secure SHell

English:
SSH is the acronym of Secure SHell and when hiring some web hosting service you can have access from your server through this protocol. Shell is a program that allows the user to have contact with the core of the operating system, through a graphical interface or through a command line such as the MS-DOS prompt on the Windows system.
Português:
SSH é a sigla de Secure SHell e ao contratar algum serviço de hospedagem de site você pode ter o acesso do seu servidor por meio deste protocolo. O Shell é um programa que possibilita que o usuário tenha contato com o núcleo do sistema operacional, por meio de uma interface gráfica ou por meio de linha de comando como era o prompt do MS-DOS, no sistema Windows.

  • Date: January 2017

Caching

Cache

English:
The cache works with a data copy that is stored to be accessed more quickly at a later time. The browser saves this data knowing that you will access that page again and so you do not have to do all the initial processes, thus saving resources and increasing the speed of data processing.
Português:
O cache funciona com uma cópia de dados que é armazenada para ser acessada mais rapidamente em um momento posterior. O navegador salva esses dados, sabendo que você acessará a página novamente e, portanto, não precisará executar todos os processos iniciais, economizando recursos e aumentando a velocidade do processamento de dados.

  • Date: January 2017
