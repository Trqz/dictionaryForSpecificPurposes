Asymmetrical

Assimétrico

English:
When graphics and text are not equal on both sides of a central line, a design is said to be asymmetrical. In the example above, there is still balance, but there are graphics on one side and text on the other.
Português:
Quando gráficos e textos não são iguais nos dois lados de uma linha central, um design é dito assimétrico. No exemplo acima, ainda há equilíbrio, mas há gráficos de um lado e texto do outro.

  • Date: January 2017

Balance

Equilibrar

English:
In design, balance involves the placement of elements on the page so that the text and graphic elements are evenly distributed. There are three ways to achieve balance: symmetrically, asymmetrically and radially.
Português:
No design, o equilíbrio envolve o posicionamento de elementos na página para que o texto e os elementos gráficos sejam distribuídos uniformemente. Existem três maneiras de alcançar o equilíbrio: simetricamente, assimetricamente e radialmente.

  • Date: January 2017

Color Schemes

Esquemas de cores

English:
The colors scheme, in color theory, is a choice of colors used in the design of a particular type of media. Web Designers use, for example, to determine how their websites colors.
Português:
O esquema de cores, na teoria das cores, é a escolha de cores usadas no design de um determinado tipo de mídia. Web Designers utilizam, por exemplo, para determinar as cores de seus websites.

  • Date: January 2017

Color Standards

Padrões de cores

English:
A set of tonalities constructed according to a specific method: some core systems are based on light nuclei, while others are based on the nuclei pigment, each system being formed by a single combination of primary tones.
Português:
conjunto de tonalidades construídas de acordo com um método específico: alguns sistemas de cores são baseados nas cores-luz, enquanto outros têm base nas cores-pigmento, com isso cada cor do sistema é formada por uma combinação única de tons primários.

  • Date: January 2017

CSS - Cascading Style Sheets

Folhas de estilo em cascata

English:
CSS is a language that describes the style of an HTML document. CSS describes how HTML elements should be displayed.
Português:
CSS é uma linguagem que descreve o estilo de um documento HTML. CSS descreve como os elementos HTML devem ser exibidos.

  • Date: January 2017

Contrast

Contraste

English:
Contrast is achieved by including elements within the design that look measurably different from one another. A designer may use color, shape, texture, size or typeface to create contrast.
Português:
O contraste é obtido pela inclusão de elementos dentro do design que parecem mensuráveis de forma diferente um do outro. Um designer pode usar cor, forma, textura, tamanho ou tipo de letra para criar contraste.

  • Date: January 2017

Grid

Grade

English:
A grid is a series of intersecting vertical, horizontal, angular or curved lines used to organize graphic elements on a page, as well as in relation to one another.
Português:
Uma grade é uma série de linhas verticais, horizontais, angulares ou curvas em interseção usadas para organizar elementos gráficos em uma página, bem como em relação à outra.

  • Date: January 2017

Gradient

Gradiente

English:
Gradient is a gradual change from one color to another. (For example, blue transitioning gradually to green).
Português:
O gradiente é uma mudança gradual de uma cor para outra. (Por exemplo, azul passando gradualmente para verde).

  • Date: January 2017

Opacity

Opacidade

English:
Opacity is synonymous with non-transparency. The more transparent an image, the lower its opacity.
Português:
Opacidade é sinônimo de não transparência. Quanto mais transparente for uma imagem, menor será sua opacidade.

  • Date: January 2017

Proximity

Proximidade

English:
The way in which design elements are grouped or spaced on a page is called proximity. Great design groups like elements together.
Português:
A maneira como os elementos de design são agrupados ou espaçados em uma página é chamada de proximidade. Grandes grupos de design, como elementos juntos.

  • Date: January 2017

Radial

Radial

English:
A radial design is one in which elements radiate from a central point, creating balance.
Português:
Um desenho radial é aquele em que os elementos irradiam de um ponto central, criando equilíbrio.

  • Date: January 2017

Repetition

Repetição

English:
To maintain a unified look, designers repeat elements throughout a design. (Repetition is also defined as the number of times your toddler asks for a cookie.)
Português:
Para manter uma aparência unificada, os projetistas repetem elementos em todo o design. (Repetição também é definida como o número de vezes que seu filho pequeno pede um cookie.)

  • Date: January 2017

Rule of thirds

Regra dos terços

English:
The rule of thirds is a technique that designers use to determine focal point. Using a grid of three rows and columns, focal points are indicated where the lines converge. Designers use this as a guide to determine where to place important elements in their design.
Português:
A regra dos terços é uma técnica que os designers usam para determinar o ponto focal. Usando uma grade de três linhas e colunas, os pontos focais são indicados onde as linhas convergem. Os designers usam isso como um guia para determinar onde colocar elementos importantes em seu design.

  • Date: January 2017

Scale

Escala

English:Scale is the size of an object in relation to another object. Scale can be used to create interest and grab a viewer’s attention.

Português:Escala é o tamanho de um objeto em relação a outro objeto. A escala pode ser usada para criar interesse e atrair a atenção do espectador.

  • Date: January 2017

Symmetrical

Simétrico

English:
Symmetry is achieved when all design elements are equal on both sides of a central line.
Português:
A simetria é alcançada quando todos os elementos de design são iguais em ambos os lados de uma linha central.

  • Date: January 2017
